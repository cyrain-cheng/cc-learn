核心设计理念

🔄
AsyncGenerator 流式架构
从 API 调用到工具执行到钩子系统，全链路使用 async function*，天然支持流式、中断、背压
🔧
工具即能力
AI 的能力边界由工具定义，每个工具是自包含模块（Schema + 逻辑 + 提示词 + 权限 + UI）
🛡️
安全内建
6 层纵深防御不是事后补丁，而是架构的一部分：钩子→权限→确认→沙箱→验证→提示词
📝
提示词即代码
通过函数动态生成，支持条件编译、运行时注入、缓存优化，直接影响 API 成本
