# cc-learn

Claude Code 源码学习笔记与解析。

## 目录结构

```
├── analysis/    # 项目分析文档（架构思想、优缺点、设计哲学等）
├── modules/     # 13 个模块的源码解析
└── front/       # 在线阅读站点（纯静态 HTML）
```

## 在线阅读

打开 `front/index.html` 或访问部署后的站点。

## 模块列表

| 编号 | 模块 | 说明 |
|------|------|------|
| 01 | 核心引擎 | queryLoop 对话循环、工具编排、流式执行 |
| 02 | 工具系统 | 40+ 内置工具的设计模式与执行流水线 |
| 03 | 提示词系统 | 系统提示词构建、缓存优化 |
| 04 | 安全与权限 | 6 层纵深防御、沙箱隔离 |
| 05 | 多 Agent 系统 | Fork/Subagent/Teammate 三种协作模式 |
| 06 | 上下文管理 | 压缩策略、记忆提取 |
| 07 | MCP 协议 | 客户端/服务器双端实现 |
| 08 | 钩子系统 | 50+ 事件类型、可扩展性设计 |
| 09 | 状态管理 | 轻量级 Store 实现 |
| 10 | 终端 UI 引擎 | 自定义 Ink、Yoga 布局 |
| 11 | 命令与技能 | 80+ 斜杠命令、插件系统 |
| 12 | API 通信层 | 流式调用、重试降级、成本追踪 |
| 13 | 辅助系统 | 认证、配置、遥测、Bridge |
